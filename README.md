# Chetna Live — AI Founder War Room

Chetna Live is a real-time AI Chief of Staff built using Gemini Live API and Google Cloud.

It transforms founder voice commands, screenshots, and live context into structured decision intelligence.

## Category
Gemini Live Agent Challenge — Live Agents

## Core Capabilities
- Real-time voice interaction
- Screenshot-based business analysis
- Decision drafting with reasoning trace
- Founder-in-loop approval control
- Multi-agent execution simulation

## Architecture Overview
Gemini Live API → FastAPI Backend → Decision Engine → Memory Layer → Google Cloud Deployment

## Tech Stack
- Gemini Live API
- Google GenAI SDK
- FastAPI
- Google Cloud Run
- Firestore (Memory Layer)

## Status
Under active development for Gemini Live Agent Challenge 2026.
